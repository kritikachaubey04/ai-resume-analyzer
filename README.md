# AI Resume Analyzer & Mock Interview Platform

A full-stack AI-powered platform that evaluates resumes and helps candidates prepare for interviews through an interactive AI mock interviewer.

## Features
- **Resume Parsing** – Automated extraction of resume content using pdfplumber
- **ATS Scoring Engine** – Evaluates resumes, identifies skill gaps, and recommends suitable job roles
- **AI Insights** – Integrates OpenAI/Gemini LLMs to extract skills and candidate insights
- **AI Mock Interviewer** – Conducts real-time interviews, generates personalized questions, and provides performance feedback

## Tech Stack
- **Frontend:** React.js, JavaScript, HTML5/CSS3
- **Backend:** FastAPI, MongoDB
- **AI/ML:** OpenAI / Gemini LLMs, pdfplumber
